# Utilizando React Hooks

## useState
  ```[state, setState] = useState(initial state)```
## useEffect: 
  * Component DidMount:
      ```useEffect(function, []) - Executado apenas uma vez, na criação do componente;```
 
  * Component DidUpdate:
      ```useEffect(function, [x]) - Executado toda vez que a variável 'x' atualizar;```
